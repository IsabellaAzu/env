
# 名刺

- 不要論もあるけど、「名刺も無いのか」と思う人も居るかもしれない
- 高齢化社会で高齢な経営者が多いのでは？
- 名刺不要の方は、写メかSNS交換かアプリかで情報交換
- 無いよりは宣伝になるのでは？
- 面白い名刺を作りたい


## 要か不要か
