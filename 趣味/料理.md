
# チャーシュー

材料

```
肉が浸るくらいで、酒1 : 醤油5
ネギの青いところ１本分
生姜薄切り数枚
```

1.沸騰したら保温６０で１時間ほど放置
2.熱を取り、冷蔵庫で冷やす
3.冷えたら薄切りにして冷凍庫へ（長期保存の場合）
